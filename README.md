# Find data type of input parameter

This repository contains a javascript code to find the data type of input parameter whether it is Number, String, Boolean, Function, Object, Array, Undefined or Null. Below is the algorithm to achieve the result.

## Algorithm

- [Step 1: Enter parameter](./pt-br)
- [Step 2: If parameter is =="null"  Return "Null is not allowed"](./en)
- [Step 3: If parameter is =="undefined" : Return "Please enter a value"](./es)
- [Step 4: If parameter typeof is "String" : Return "String"](./ne)
- [Step 5: If parameter typeof is "Number" : Return "Number"](./he)
- [Step 6: If parameter typeof is "Boolean" : Return "Boolean"](./id)
- [Step 7: If parameter instanceOf is "Object" and instanceOf is "Array" :  Return "Array"](./ko)
- [Step 8: If parameter instanceOf is "Object" and Not instanceOf is "Array" : Return "Object""](./id)
- [Step 9: If parameter typeof is "Function"  Return "Function"](./ko)
